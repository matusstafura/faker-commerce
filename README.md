# PHP Faker eCommerce

## Usage

```php
// create lang en_EN by default
$ff = FakerFaktory::create();

// breadcrumbs with separator "/" by default
$ff->breadcrumbs()->level(4);
// Home / Sport / Outdoor / Jerseys

Create Category // see table for all avalailable Categories
$ff->category('Sport');
// 'Soccer'

$ff->product();
```
